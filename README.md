# Дипломный проект по теме API на курсе "Автоматизатор тестирования на Java"

### Технологии проекта:

Java 11, Junit 4.13.2, Maven 3.8.1, Allure 2.10.0, Postman, Rest Assured 5.3.0

### Как запускать:

`mvn clean test`
